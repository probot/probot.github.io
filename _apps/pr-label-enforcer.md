If you want to require certain labels before your PR can be merged, just enable this bot and add a config file under at `.github/pr_labels.yml` here's a sample config file:
```yaml
version: '1'
invalidStatus: "pending"
labelRule:
  endsWith:
    - "g"
  values:
    - "bug"
    - "enhancement"
```
